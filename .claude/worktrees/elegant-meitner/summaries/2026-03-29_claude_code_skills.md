## 🎯 ¿Qué es?
Cómo aprovechar al máximo Claude Code mediante el uso de habilidades especializadas en lugar de peticiones genéricas.

## 🚀 Puntos Clave
- **Organización**: Estructura `.claude/skills/` que usa archivos `skill.md` para inyectar contexto bajo demanda.
- **Eficiencia**: Ahorro masivo de tokens al evitar que el agente lea todo el repo sin necesidad.
- **Herramientas**: Plugin oficial **Skill Creator** para orquestar los agentes.

## 💼 Oportunidad Workflow Waves
- **Problema real**: Agentes de IA que se vuelven "tontos" o caros al tener demasiado contexto innecesario.
- **Pitch**: "Optimizamos tus flujos de IA con una arquitectura de 'Skills' que hace a los agentes más rápidos, precisos y un 60% más baratos de operar".

## ⚖️ ¿Vale la pena automatizar?
- **ROI**: **ALTO.** Mejora la escalabilidad técnica de la agencia. Menos errores en código = entregas más rápidas a clientes.
