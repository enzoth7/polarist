## 🎯 ¿Qué es?
Mapeo comparativo de las 3 herramientas principales de Anthropic para decidir el entorno adecuado.

## 🚀 Puntos Clave
- **Claude AI**: Para lluvia de ideas, planes estructurados y contenidos largos. **Pro**: Multidispositivo. **Contra**: Sin acceso local.
- **Claude Code**: Para construcción y depuración de bases de código completas en terminal. **Pro**: Contexto de 1M de tokens, CI/CD integrado. **Contra**: Curva de aprendizaje técnica.
- **Claude Cowork**: El agente de escritorio (GUI). Automatiza clics y tareas repetitivas en el sistema de archivos real sin programar. **Pro**: Automatización nativa de archivos.

## 💡 ¿Cómo Usarlo?
- **Ideas** -> Claude AI
- **Construir** -> Claude Code
- **Automatizar Escritorio** -> Claude Cowork
