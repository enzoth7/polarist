## 🎯 ¿Qué es?
Una comparativa de los tres sabores actuales de Claude para diferentes perfiles y casos de uso.

## 🚀 Puntos Clave
- **Claude Code**: Herramienta CLI autónoma. Ideal para refactorización profunda y corrección de bugs en múltiples archivos desde la terminal.
- **Claude Cowork**: App de escritorio para perfiles no técnicos. Se enfoca en automatizar gestión de archivos, extracción de datos de PDFs y flujos entre aplicaciones.
- **Claude AI**: El asistente conversacional estándar para borrador de textos e investigación general.

## 💼 Oportunidad Workflow Waves
- **Problema real**: Clientes no técnicos (contadores, abogados, pymes) que necesitan IA pero no quieren aprender Python ni usar terminales.
- **Pitch**: "Te configuramos un entorno **Claude Cowork** optimizado para tu despacho que automatiza la lectura de documentos legales y financieros sin que tengas que programar".

## ⚖️ ¿Vale la pena automatizar?
- **ROI**: **MEDIO.** No requiere automatización compleja de nuestra parte, sino configuración experta. El tiempo ahorrado es directamente para el cliente final, lo que es un servicio de implementación de bajo coste y alto valor.
