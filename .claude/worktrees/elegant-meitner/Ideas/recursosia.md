## 🧠 Estrategia de Modelos AI (API vs Web Subscriptions)
**Decisión Estratégica Fundamental (Marzo 2026):**
- **ChatGPT Plus / Claude Pro (Suscripciones Web de $20/m):** Tienen límites duros de mensajes, se saturan rápido con contexto largo, y **NO TIENEN SOPORTE MCP (Model Context Protocol)**. No pueden conectarse localmente a n8n, bases de datos o archivos del sistema.
- **La Solución (Pago por Uso - API):** Usar interfaces locales (como Antigravity, Cursor, o Claude Code) conectadas a plataformas como **OpenRouter** o la API directa. 
    - *Ventajas:* Sin límites arbitrarios de mensajes (solo límite de recarga de saldo), herramientas MCP nativas (vital para automatizaciones en n8n o leer archivos locales), y cobro exacto por token consumido.
    - *Veredicto:* Para trabajo profesional e integraciones, la API + Agente Local vence rotundamente a la suscripción web.
