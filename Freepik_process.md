# PROCESO EN FREEPIK    

Usamos PROMPT 1 y PROMPT 2 con una imagen real del producto que nos manda la marca, más una imagen de referencia que deberia estar en base a la marca (estilo, tipogracia, tipo de cliente, nicho, etc).
Todo esto lo hacemos para que en Freepik nos de un prompt armado en base a estos 4 elementos (PROMPT 1, PROMPT 2, PRODUCTO, REFERENCIA) y eso crea el prompt final para crear la imagen mejorada.
Con la imagen mejorada, introducimos "Referencia estilo para el anuncio" + Concepto del anuncio que se quiere hacer (en este caso la imagen mejorada con el anuncio) donde se escribe, segun el estilo de la marca, que tipo de anuncio se quiere.


## PROMPT 1 
The image connected to the node labeled “PRODUCTO” contains the client’s actual product.
The image connected to the node labeled “REFERENCIA” defines the target aesthetic, environment, lighting, composition, and overall editorial style.
Your task:
Replace the product visible in the “REFERENCIA” image with the product shown in the “PRODUCTO” node.
Maintain exactly:
Same environment
Same lighting direction, softness, and temperature
Same camera angle and lens logic
Same framing and composition
Same depth of field
Same mood and emotional tone
Same materiality and surface behavior
Only adapt:
Correct physical scale of the new product
Realistic weight and contact points
Natural integration into the scene
Logical interaction with environment or avatar (if present)
The final result must look as if the product from “PRODUCTO” was originally photographed inside the environment of “REFERENCIA”.
No aesthetic reinterpretation.
No stylistic changes.
No generic AI look.
No floating objects.
No render appearance.
Preserve editorial realism, micro-textures, and physical coherence.
If a person appears in the reference:
Keep identical lighting logic
Maintain anatomical realism
Ensure natural skin texture
Preserve believable physical interaction
The output must feel like a real commercial editorial photograph — not a generated composite.

## PROMPT 2
🧠 PROMPT CONSTRUCTION SYSTEM – STRUCTURED MODE
From now on, every generated prompt must strictly follow this strategic and visual architecture.
Do not generate generic prompts.
Do not improvise aesthetics.
Do not use vague terms.
🔴 STEP 1 — STRATEGIC DEFINITION (BEFORE WRITING THE PROMPT)
Before describing the image, internally define:
Perceived socioeconomic level (NSE positioning)
One dominant human motivation (maximum 2 secondary)
Dominant mental trigger (button)
Key fear that must be visually neutralized
Core benefit that must be felt (not explained)
Cultural visual code
Final emotional state after viewing the image
If these are not clearly defined → the prompt is incorrectly constructed.
The image must be able to sell without text.
🎬 STEP 2 — MANDATORY PROMPT STRUCTURE
The prompt must be written in this exact order:
Type of image (editorial product shot / commercial campaign image)
Product (precise physical description)
Scenario (detailed environment + defined materiality)
Lighting (type, direction, temperature, emotional intention)
Camera (lens, aperture, realistic depth of field, framing type)
Real textures and controlled imperfections (anti-render look)
Frozen motion elements if applicable (micro particles, tension, condensation, fabric movement)
Clear and singular emotional mood
Technical restrictions:
🧱 NON-NEGOTIABLE REALISM RULES
Nothing can float.
Light must have coherent direction.
Surfaces must show real micro-texture.
Nothing should look overly perfect or plastic.
Avoid flat frontal angles. Prefer subtle editorial perspectives (slight diagonal, soft Dutch tilt, partial crop).
If it looks like a 3D render → commercial failure.
👤 IF AN AVATAR IS INCLUDED
Apply these rules:
The product is the hero.
The avatar is contextual support, not the protagonist.
Perfect lighting coherence between skin, product, and environment.
Natural skin texture (visible pores, subtle imperfections).
Expression aligned with the strategic emotional goal.
Styling consistent with the perceived socioeconomic level.
Real physical interaction (logical weight, finger tension, believable posture).
Editorial framing, not obvious advertising pose.
If the avatar competes with the product → conversion drops.
🔒 TYPOGRAPHY HARD-LOCK (IF PRODUCT CONTAINS LABELS OR TEXT)
Original label preserved exactly as is — same typography, same font, same font weight, same kerning, same spacing, same layout, same hierarchy.
Do not modify, reinterpret, stylize, translate, regenerate, correct, or improve any text.
All words, numbers, symbols, punctuation marks, percentages, measurements and special characters must remain identical to the original.
No typography or symbol changes of any kind.
This block is non-editable and mandatory.
🧠 FINAL VALIDATION CHECK
Before delivering the prompt, verify:
Does it activate one clear dominant emotion?
Does it reduce mental friction?
Does it feel natural rather than pushy?
Could it sell without text?
Does it look like a real editorial photograph?
If any answer is uncertain → rewrite.
